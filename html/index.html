Dockerfile per Image

FROM centos:latest

ENV JAVA_HOME /usr/lib/jvm/java-1.8.0-openjdk/bin

RUN yum install -y -q java-1.8.0*

Build the image:

docker build -t akrieger/java:8-centos -t akrieger/java:latest .

Dockerfile Syntax

  • FROM - Starting image
  • COPY - Files for the new image
  • VOLUME - Data
  • EXPOSE - Ports
  • USER/ENV
  • RUN - Runs/Compiles
